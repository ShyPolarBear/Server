## ✏️ Main Feature

- jwt/OAuth을 이용한 로그인

## 🔧 Stack

- **Language** : java17
- **Library & Framework** : Spring boot, Spring Security, Spring Data jpa, Querydsl
- **Database** : MySQL
- **ORM** : Jpa
- **Deploy** : AWS(EC2, S3, RDS), Docker
- **CI/CD** : Github Actions
